# Agents and Skills Talk Repo

This folder is a lightweight, talk-ready repo scaffold modeled after an `agents-and-skills` project setup.

## Included assets

- `slides/quotes-and-focuses.md` — slide content for:
  - quote callouts
  - itemized focus areas
- `visualization/progress-path.svg` — static timeline graphic for progress path updates
- `visualization/progress-path.json` — source timeline data with dates and notes

## Suggested workflow

1. Update quote and focus slides in `slides/quotes-and-focuses.md`.
2. Add new milestones to `visualization/progress-path.json`.
3. Regenerate or manually edit `visualization/progress-path.svg` as needed for the latest talk revision.
